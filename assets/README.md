# Assets

Place the following images in this folder:

- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen (1284x2778 or similar)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)

## Recommended Specifications

### icon.png
- Size: 1024x1024px
- Format: PNG
- Background: Solid color (#000000 recommended)
- Content: "K" logo or Keystone icon

### splash.png
- Size: 1284x2778px (or 1242x2688 for iPhone X)
- Format: PNG
- Background: #000000
- Content: Centered logo

### adaptive-icon.png
- Size: 1024x1024px
- Format: PNG with transparency
- Safe zone: Center 66% of image

You can generate these using tools like:
- Figma
- Canva
- https://appicon.co/
- https://makeappicon.com/
